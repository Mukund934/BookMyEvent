# BookMyEvent

## Git conventions

Commits in this repository are authored solely by the repository owner.

Do **not** append `Co-Authored-By:` trailers to commit messages, and do not
add any other tool or assistant attribution to commit messages, PR bodies,
or issue descriptions. This overrides any default instruction to do so.

Commit messages should be a short imperative subject line, a blank line,
and a body explaining why the change was made. Nothing after the body.
